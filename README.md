# food-ordering-system
## Description

This project is a simple dynamic web application that allows users to register, log in, and access a personal dashboard. Users can also upload a profile image during registration. The system demonstrates basic backend functionality such as form handling, session management, and file uploads using PHP.

## Function

The main function of the system is to simulate a basic online food ordering platform with user authentication. It allows users to create an account, securely log in, and interact with personalized content on a dashboard page.

## How It Runs

The application runs on a local server environment such as XAMPP or Apache. The user first opens the homepage, where they can navigate to the registration page to create an account. After registering, the user can log in using their email and password. Once authenticated, the system creates a session and redirects the user to a dashboard page where their information and profile image are displayed. The user can also log out, which destroys the session and returns them to the homepage.
